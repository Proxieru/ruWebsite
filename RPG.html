Documenting Memory Cheat Codes

00:3032 (Character slots empty)

Findings: If you set it to 1, you can only play as Mario, same for settings it to 2 or 3.
If set beyond 5 (the normal max amount that can be in your party) it will glitch out your switch menu and elements of the switch menu will overlap.

00:3033-00:3037 (Character slots 1-5)

Findings: If set beyond normal amount of characters, for example if set to 6 it will put a bugged character into whatever slot you chose.
For 6, the character name is Fire with a star before the name, possibly reading from enemy spell memory? It also uses the icon for the Frog Sage's staff.
And is also level 15 with glitched characters in front of the HP value which is 35/35. Another thing about these bugged characters is that they often use different sprites in battle rather than the menu.
And have no animations. Some are unstable and will freeze the game if you attack an enemy, but some don't. Also, they sometimes have Dumm or Dummy attacks which usually don't do anything or will freeze the game on use. Some also have glitched items such as an item named "Weapon" as an example.
If you want to find more glitchy sprites/characters or behavior, I encourage you to grab an emulator, type in the cheat code like: 3033=(Number) and try it out! (This should work with the other codes too, so try them out!)

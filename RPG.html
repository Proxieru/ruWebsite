Environment Based Addresses

0x003030 - Current Area
Doesn't seem to affect the current area your in but if set to any value (Besides 38, which does something different) you lose the ability to enter loading zones, but with doors you can noclip a bit above the door. anyways back to the case I mentioned with the value 38, if you enable this and you enter a loading zone (doors count as well) it will display the message "This is your last chance" which is from Bowser's Keep (after the bridge collapses) where you have to complete challenges within a certain amount of tries (you still cannot enter loading zones and doors) also if you warp to anywhere from the map you will end up in a black void and cannot move
Character Based Addresses
0x003032 - Character Slots Open (1-5)
Normally is restricted to the values [1-5] as that is the normal amount of characters that can be apart of your party in the game, but if you try to set it below 0 and try to open the menu the game freezes [same with negative values]. anyways besides that, if set outside the normal range [5-10] in the switch menu, the character menu will overlap with the text "Switch", "Which two?" and you can get two characters such as having two identical Mario's and such (Is quite funny actually) the game also freezes if you go beyond 10.

How to use

Grab a emulator, go into the cheat menu of said emulator and type in any of the provided memory address. for example: 0x003037=[value] and then add the cheat and enable it and your done!
